# ChanDaLiar
Connect OpenAI with Azure Cognitive services and create the speaking ChanDaLiar.
Hosted demo can be viewed here https://chan-da-liar.vercel.app/

## Start development
```
npm install
npm start
```


## Generate code

### Services
```
npx ng g s <name>
```

### Components
```
npx ng g c <name>
```
